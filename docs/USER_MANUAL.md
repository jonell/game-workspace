## 1. 系统概述

### 1.1 系统简介

蠢驴电竞陪玩派单管理系统是一套专为电竞陪玩工作室设计的综合管理平台。系统覆盖从客户管理、订单创建与分配、陪玩接单执行，到报账审核、收入统计的全业务流程，帮助工作室实现高效、规范的运营管理。

### 1.2 技术架构

```
浏览器 (React) ──HTTP──▶ Nest.js 后端 ──▶ PostgreSQL + Redis
                                  △
陪玩桌面端 (Electron 客户端) ──WebSocket──┘
```

- **前端**：React + Ant Design 组件库，响应式 Web 界面
- **后端**：Nest.js (Node.js)，提供 RESTful API + WebSocket 实时通信
- **数据库**：PostgreSQL 16，使用 Prisma ORM
- **缓存**：Redis 7
- **陪玩客户端**：Go 语言编写的桌面代理程序，通过 WebSocket 与服务器保持长连接

### 1.3 四种用户角色

| 角色 | 标识 | 职能概述 |
|------|------|----------|
| **老板 (OWNER)** | OWNER | 查看盈亏统计、管理客户、管理员工、管理工作室、审核客服/陪玩注册 |
| **管理员 (ADMIN)** | ADMIN | 管理所有订单、管理陪玩、管理客户、审核报账、远程控制陪玩电脑 |
| **客服 (CS)** | CS | 在派单工作台创建订单、查看订单池与陪玩状态、查看历史派单记录 |
| **陪玩 (COMPANION)** | COMPANION | 通过桌面客户端接单/确认/完成订单、切换工作/娱乐模式、提交报账 |

## 2. 快速入门

### 2.1 登录系统

1. 在浏览器中打开系统地址（由管理员提供）
2. 进入登录页面，页面顶部显示「蠢驴电竞陪玩派单管理系统」
3. 输入**用户名**和**密码**
4. 点击「登录」按钮
5. 登录成功后，系统会根据你的角色自动跳转到对应的工作页面

登录后进入主界面，分为三个区域：

- **左侧导航菜单**：根据角色显示对应的功能菜单，可点击切换功能页面。左侧菜单栏可折叠/展开（点击左上角按钮）。

1. **老板登录**：使用 OWNER 账号登录

4. **授权员工**：CS 和 COMPANION 角色需要授权后才能登录。进入「客户端授权」页面，找到待审核的用户，点击「审核通过」
5. **为陪玩配置客户端**：COMPANION 登录后需下载并运行 Electron 客户端 桌面客户端

## 3. 角色操作指南：老板 (OWNER)

老板拥有系统的最高权限，主要负责查看经营数据和团队管理。

### 3.1 盈亏统计

> 路径：左侧菜单 → **盈亏统计**

此页面用于查看工作室的盈亏数据，包含收入统计、支出记录和利润图表。

**操作步骤：**

1. 点击「盈亏统计」菜单进入页面
2. 系统会弹出**二次密码输入框**，请输入二次密码（默认：`888888`）
3. 验证通过后，页面展示盈亏汇总数据
4. 可切换日视图/月视图查看不同维度的统计
5. 图表区域直观展示收支趋势

> **安全提示**：二次密码验证有效期为 5 分钟，超时后需重新输入。

### 3.2 客户管理

> 路径：左侧菜单 → **客户管理**

**查看客户列表：**

表格展示所有客户的以下信息：

| 列名 | 说明 |
|------|------|
| 客户编号 | 系统自动生成的唯一编号（格式：C + 随机码） |
| 微信号 | 客户的微信号 |
| 平台 | 沟通平台（微信/QQ/电话/其他） |
| 陪玩 | 当前分配的陪玩（未分配则显示"未分配"） |
| 累计消费 | 该客户的总消费金额 |
| 操作 | 编辑、删除按钮 |

**新建客户：**

1. 点击「新建客户」按钮
2. 在弹出的表单中填写：
   - **微信号**（必填）
   - **平台**（必选，下拉选择：微信/QQ/电话/其他）
   - **平台账号**（选填）
   - **备注**（选填）
3. 点击「保存」

**编辑客户：**

1. 在表格中找到目标客户，点击「编辑」按钮
2. 修改需要更新的字段
3. 点击「保存」

**删除客户：**

1. 点击「删除」按钮
2. 在确认弹窗中点击「确定」完成删除

### 3.3 员工管理

> 路径：左侧菜单 → **员工管理**

此页面用于创建和管理系统内的所有员工账号（ADMIN、CS、COMPANION）。

**新建员工：**

1. 点击「新建员工」按钮
2. 填写：
   - **用户名**（必填，唯一）
   - **密码**（必填）
   - **角色**（必选：管理员/客服/陪玩）
   - **所属工作室**（必选）
   - 如角色为「陪玩」，还需填写：
     - **报账编号**：陪玩的唯一结算编号
     - **擅长游戏**：多选，如"英雄联盟"、"王者荣耀"
     - **分成比例**：默认为 0.6（即 60%）
3. 点击「保存」

**编辑/删除员工：** 在表格操作列点击对应按钮即可。

**重置密码：** 在员工详情或编辑弹窗中可重置该员工的登录密码。

### 3.4 工作室管理

> 路径：左侧菜单 → **工作室管理**

管理工作室的创建、编辑和删除。

**新建工作室：**

1. 点击「新建工作室」
2. 输入工作室名称
3. 点击「保存」

每个工作室下可包含多个员工、陪玩和客户，数据按工作室隔离。

### 3.5 客户端授权

> 路径：左侧菜单 → **客户端授权**

客服 (CS) 和陪玩 (COMPANION) 角色注册后需要经过授权才能登录系统。

**操作步骤：**

1. 进入「客户端授权」页面，查看待授权用户列表
2. 表格展示：
   - 用户名
   - 角色
   - 所属工作室
   - 注册时间
   - 审核状态（待审核/已通过）
3. 对于待审核的用户，点击「审核通过」
4. 审批后该用户即可正常登录

> **拒绝授权**：如果拒绝，该用户将无法登录系统。

## 4. 角色操作指南：管理员 (ADMIN)

管理员负责日常运营管理，是系统使用最频繁的角色。

### 4.1 派单管理

> 路径：左侧菜单 → **派单管理**

此页面与客服的「派单工作台」功能类似，可查看所有订单、创建订单、指定陪玩。

**查看全部订单：**

- 表格按创建时间倒序显示所有订单
- 可筛选订单状态
- 包含订单号、客户、游戏、金额、派单方式、陪玩、状态等列

**创建订单：**

1. 点击「创建订单」
2. 填写订单信息（详见 [5.1 派单工作台](#51-派单工作台)）
3. 点击「创建」

**指派陪玩：** 点击订单的「指定陪玩」，从在线空闲陪玩中选择，确认即可。

**取消订单：** 对于待处理状态 (PENDING) 的订单，可点击「取消」。

### 4.2 陪玩管理

> 路径：左侧菜单 → **陪玩管理**

查看所有陪玩的状态和工作情况。

**表格内容：**

- **陪玩姓名**：关联用户名
- **状态**：在线/空闲/忙碌/离线
- **当前模式**：工作模式/娱乐模式
- **擅长游戏**：标签展示
- **在线时长**：当日累计在线时间
- **接单数**：当日完成订单数

### 4.3 客户管理

> 路径：左侧菜单 → **客户管理**

功能与老板的客户管理相同，额外支持**重新分配陪玩**功能。

**重新分配陪玩：**

1. 在客户列表中找到目标客户
2. 点击操作列的「分配陪玩」
3. 从陪玩列表中选择新的陪玩
4. 确认后，该客户的所有新订单将优先派给新分配的陪玩

### 4.4 报账审核

> 路径：左侧菜单 → **报账审核**

这是管理员最核心的工作之一。陪玩完成订单后提交报账，管理员在此审核。

**三栏标签页：**

页面顶部显示三个统计卡片：待审核数量、已通过数量、已拒绝数量。

下方有三个标签页：

| 标签页 | 说明 |
|--------|------|
| **待审核** | 陪玩刚提交、等待审核的报账记录 |
| **已通过** | 已审核通过的记录 |
| **已拒绝** | 已审核拒绝的记录 |

**表格字段：**

| 列名 | 说明 |
|------|------|
| 陪玩 | 提交报账的陪玩姓名 |
| 订单号 | 关联的订单编号 |
| 金额 | 报账金额（红色加粗显示） |
| 支付方式 | 如微信、支付宝等 |
| 截图 | 支付凭证截图缩略图，点击可放大查看 |
| 支付时间 | 实际支付时间 |
| 提交时间 | 报账提交时间 |
| 状态 | 待审核/已通过/已拒绝 |
| 操作 | 通过/拒绝按钮 |

**审核单条报账：**

1. 在「待审核」标签页中，查看报账详情
2. 点击截图缩略图可查看完整支付凭证
3. 确认无误后点击「通过」
4. 如发现问题（金额不符、截图不清等），点击「拒绝」

**批量操作：**

1. 勾选多条待审核记录前的复选框
2. 页面上方出现蓝色操作栏，显示已选数量
3. 点击「批量通过」或「批量拒绝」
4. 在确认弹窗中点击「确定」

> **提示**：批量操作仅在「待审核」标签页可用。切换标签页会自动清除已选记录。

### 4.5 进程黑名单管理

> 路径：左侧菜单 → **进程黑名单** / **进程白名单** / **杀进程日志**

功能：
- 管理陪玩终端禁止运行的软件（如外挂、竞品平台）
- 添加支持两种方式：从陪玩上报进程多选 / 手动输入
- 白名单优先于黑名单（白名单进程不被关闭）
- 22 个系统内置白名单（微信、浏览器等）

操作：
1. 添加进程 → 选择模式 → 多选/手动输入
2. 推送黑名单 → 全工作室或指定陪玩
3. 终端收到后 5 秒倒计时弹窗 → 自动关闭进程
4. 杀进程日志可查看历史

权限：ADMIN / OWNER / CS 均可

### 4.6 远程控制

> 路径：左侧菜单 → **远程控制**

管理陪玩电脑的远程操作，前提是陪玩的 Electron 客户端 已在线。

**功能说明：**

| 列名 | 说明 |
|------|------|
| 陪玩姓名 | 关联用户名 |
| 整体状态 | 在线/空闲/忙碌/离线 |
| PC 状态 | 电脑是否在线（绿色=在线，灰色=离线） |
| 心跳时间 | 最后一次心跳包时间，2 分钟内为在线 |
| 当前模式 | 工作模式 / 娱乐模式 |
| 限速状态 | 是否限速及限速值 |
| 操作 | 关机/重启/限速/解除限速 |

**执行远程命令：**

1. 确认陪玩的 PC 状态为「在线」
2. 点击对应操作按钮：
   - **关机**：发送关机指令到陪玩电脑（危险操作，需确认）
   - **重启**：发送重启指令
   - **限速**：限制陪玩电脑网络速度为 500KB/s（防止娱乐模式影响工作）
   - **解除限速**：移除网速限制
3. 在弹出的确认窗口中点击「确定」
4. 系统显示指令发送成功

> **注意**：只有 PC 在线（心跳正常）时才能执行远程命令。离线状态的按钮为灰色不可点击。

## 5. 角色操作指南：客服 (CS)

客服是派单系统的核心操作者，负责创建订单和跟踪订单状态。

### 5.1 派单工作台

> 路径：左侧菜单 → **派单工作台**

派单工作台分为三栏布局：

#### 左栏：陪玩状态

实时显示当前工作室所有陪玩的状态：

| 状态 | 标签颜色 | 说明 |
|------|----------|------|
| 在线 (ONLINE) | 红色 | 陪玩在线但未设置为空闲 |
| 空闲 (IDLE) | 绿色 | 陪玩空闲，可接收订单 |
| 忙碌 (BUSY) | 金色 | 陪玩正在处理订单 |
| 离线 (OFFLINE) | 灰色 | 陪玩未登录客户端 |

列表每 10 秒自动刷新。

#### 中栏：订单池

显示所有待派发的池子订单（POOL 模式、状态为 PENDING）。

每个订单卡片展示：
- 游戏名称
- 订单类型标签（新单/续费/复购/打赏）
- 客户微信号
- 订单金额
- 「指定陪玩」按钮

> 订单池也是每 10 秒自动刷新。

点击「指定陪玩」按钮，弹出陪玩选择窗口：
- 下拉列表仅显示在线或空闲的陪玩
- 可搜索陪玩姓名
- 选择后点击「确认派单」，订单将直接指派给该陪玩

#### 右栏：快捷统计

实时显示三个关键指标：
- **可接单陪玩**：在线 + 空闲的总人数（绿色）
- **忙碌中**：正在处理订单的人数（金色）
- **待派订单**：订单池待派发的订单数（蓝色）

#### 创建订单

点击右上角「创建订单」按钮，弹出订单表单：

| 字段 | 必填 | 说明 |
|------|------|------|
| 订单类型 | 是 | 下拉选择：新单/续费/复购/打赏 |
| 游戏名称 | 是 | 输入游戏名称，如"英雄联盟" |
| 金额 | 是 | 输入数字，支持两位小数，前缀 ¥ |
| 派单方式 | 是 | **入池抢单**：订单放入池中，陪玩自行抢单<br>**指定派单**：直接分配给指定陪玩 |
| 客户 | 否 | 搜索选择已有客户（按微信号搜索） |
| 时长 | 否 | 预计服务时长（小时），步长 0.5 |

填写完成后点击「创建」按钮，订单即创建成功：
- 选择「入池抢单」的订单出现在中栏的订单池
- 选择「指定派单」的订单直接发送给指定陪玩的客户端

### 5.2 派单记录

> 路径：左侧菜单 → **派单记录**

查看自己创建的所有订单历史记录。

- 表格按创建时间倒序排列
- 支持按订单状态筛选（下拉选择）
- 包含完整订单信息：订单号、客户、游戏、金额、派单方式、陪玩、状态、创建时间

### 5.3 陪玩状态

> 路径：左侧菜单 → **陪玩状态**

全屏查看所有陪玩的实时状态，便于快速判断可派单人员。

- 列表显示陪玩姓名、状态标签、擅长游戏、当前模式等
- 支持手动刷新

## 6. 角色操作指南：陪玩 (COMPANION)

陪玩不需要通过 Web 浏览器工作，而是使用专用的 Electron 客户端 桌面客户端。

### 6.1 安装与启动 Electron 客户端

1. 从管理员处获取 Electron 客户端 可执行文件
2. 配置环境变量：
   - `AGENT_SERVER_URL`：后端服务器地址（默认 `http://localhost:3001`）
   - `AGENT_TOKEN`：你的登录 Token
3. 运行 Agent 程序
4. 启动后 Agent 自动连接服务器，保持 WebSocket 长连接

### 6.2 本地 Web 界面

Agent 启动后，在浏览器中打开 `http://localhost:9876` 可以看到本地控制面板：

**状态显示：**
- 当前模式（工作/娱乐）
- 工作时间计时
- 娱乐时间计时
- 总在线时长

**模式切换：**

- **工作模式 (WORK)**：切换后状态变为「在线 (ONLINE)」，可以接单
- **娱乐模式 (ENTERTAINMENT)**：切换后状态变为「空闲 (IDLE)」，暂时不接单

模式切换通过点击按钮完成，Agent 会立即同步状态到服务器。

### 6.3 订单通知与处理

当有新订单指派给你时，Agent 会通过 WebSocket 实时接收通知，本地界面会显示订单信息。

**订单处理流程：**

1. **收到订单**：本地界面弹出订单通知，显示游戏名称、金额、客户信息
2. **抢单**（池子订单）：在订单池中看到感兴趣的单子，可通过 Agent 点击「抢单」
3. **确认接单**：点击「确认」按钮，订单状态变为 CONFIRMED（已确认）
4. **开始服务**：确认后开始为玩家提供陪玩服务
5. **完成订单**：服务完成后，点击「完成」按钮，订单状态变为 DONE（已完成）

### 6.4 报账提交

完成订单后，需要在系统中提交报账：

1. 进入报账页面
2. 关联已完成订单
3. 填写报账金额
4. 选择支付方式
5. 上传支付截图
6. 填写实际支付时间
7. 提交报账

提交后报账进入「待审核」状态，等待管理员审核。

### 6.5 远程控制说明

管理员可通过「远程控制」页面对你的电脑执行以下操作：
- 关机/重启
- 网络限速/解除限速

当收到远程指令时，Agent 会自动执行。请确保在关键时刻保存好工作。

## 7. 核心工作流

### 7.1 订单全生命周期

```
客服创建订单
    │
    ├── 入池抢单 (POOL) ──▶ 订单进入订单池
    │                           │
    │                    ┌──────┴──────┐
    │                    │             │
    │              陪玩抢单 (grab)   客服指定 (assign)
    │                    │             │
    │                    └──────┬──────┘
    │                           │
    └── 指定派单 (DIRECT) ──────┘
                │
          状态: GRABBED
                │
          陪玩确认接单 (confirm)
                │
          状态: CONFIRMED
                │
          陪玩完成服务 (complete)
                │
          状态: DONE
                │
          陪玩提交报账
```

**状态转换规则：**

| 当前状态 | 可转换到 |
|----------|----------|
| PENDING（待处理） | GRABBED（已抢单）、CANCELLED（已取消） |
| GRABBED（已抢单） | CONFIRMED（已确认）、CANCELLED（已取消） |
| CONFIRMED（已确认） | DONE（已完成）、CANCELLED（已取消） |
| DONE（已完成） | 不可转换（终态） |
| CANCELLED（已取消） | 不可转换（终态） |

**订单类型说明：**

| 类型 | 标识 | 说明 |
|------|------|------|
| 新单 | NEW | 新客户首次下单 |
| 续费 | RENEW | 老客户续费 |
| 复购 | REPURCHASE | 老客户再次购买 |
| 打赏 | TIP | 客户额外打赏 |

### 7.2 报账审核流程

```
陪玩完成订单 (DONE)
    │
陪玩提交报账 (含截图)
    │
报账状态: PENDING（待审核）
    │
    ├── 管理员审核通过 ──▶ APPROVED（已通过）
    │                          │
    │                    金额计入收入流水
    │
    └── 管理员拒绝 ──────▶ REJECTED（已拒绝）
                               │
                         陪玩可重新提交
```

**报账状态：**

| 状态 | 说明 |
|------|------|
| PENDING | 待审核 |
| APPROVED | 审核通过，已计入收入 |
| REJECTED | 审核拒绝 |

### 7.3 收入追踪流程

```
报账审核通过
    │
    └── 按日/按月汇总统计
            │
    ┌───────┴────────┐
    │                │
 日收入报表       月收入报表
 (按订单类型)     (按陪玩排名)
    │                │
    └───────┬────────┘
            │
    ┌───────┴────────┐
    │                │
  页面查看        CSV 导出
```

## 8. 常见问题 (FAQ)

### 账号与登录

**Q: 登录时提示"用户名或密码错误"怎么办？**

检查用户名和密码是否正确。如有疑问联系管理员重置密码。

**Q: 登录时提示"账号尚未通过审核，请联系管理员"？**

CS 和 COMPANION 角色的账号需要由老板审批后才能登录。请联系老板在「客户端授权」页面进行审批。

**Q: 忘记二次密码怎么办？**

联系老板在后台重置二次密码。当前版本不支持自助找回。

**Q: 登录后能同时打开多个页面吗？**

可以。Token 存储在浏览器的 sessionStorage 中，同一浏览器新标签页需重新登录。

**Q: 登录后多久会自动退出？**

Access Token 有效期为 15 分钟，Refresh Token 有效期为 7 天。Token 过期后自动跳转登录页。

### 订单与派单

**Q: "入池抢单"和"指定派单"有什么区别？**

- **入池抢单**：订单进入公共订单池，所有空闲陪玩都可以看到并抢单，先到先得
- **指定派单**：客服或管理员直接将订单分配给指定的陪玩，其他陪玩看不到

**Q: 订单可以取消吗？**

只有状态为 PENDING（待处理）或 GRABBED（已抢单）的订单可以取消。已确认或已完成的订单不可取消。

**Q: 为什么我看不到某些订单？**

不同角色看到的数据范围不同：
- 客服只能看到自己创建的订单
- 陪玩只能看到分配给自己或自己抢的订单
- 管理员可以看到本工作室所有订单

**Q: 池子里的订单为什么没有陪玩接？**

确认是否有空闲（IDLE 或 ONLINE 状态）的陪玩在线。陪玩需要在 Agent 客户端中切换到「工作模式」才能接单。

### 报账与收入

**Q: 报账审核被拒绝后可以重新提交吗？**

可以。被拒绝的报账可以修正后重新提交。

**Q: CSV 导出文件乱码怎么办？**

导出的 CSV 使用 UTF-8 编码并包含 BOM 头。在 Excel 中如果乱码，请使用「数据 → 从文本/CSV 导入」功能，选择 UTF-8 编码。

### 陪玩客户端 (Electron 客户端)

**Q: Electron 客户端 显示离线怎么办？**

1. 检查 Agent 程序是否正在运行
2. 确认网络连接正常，能访问后端服务器
3. 检查 `AGENT_TOKEN` 配置是否正确
4. 重启 Agent 程序

**Q: 切换娱乐模式后为什么还能收到订单？**

娱乐模式下状态为 IDLE（空闲），管理员仍可主动为你指派订单。如需完全拒绝新单，请退出 Agent。

**Q: 限速会影响游戏体验吗？**

限速 (500KB/s) 可能会影响在线游戏的网络表现。管理员通常在娱乐模式时使用限速，工作模式前会解除限速。

### 系统与权限

**Q: 不同角色能看到对方的数据吗？**

系统按工作室和角色进行数据隔离：
- OWNER 可查看所有数据
- ADMIN 可查看本工作室的所有数据
- CS 只能看到自己创建的订单
- COMPANION 只能看到与自己相关的订单

**Q: 如何修改员工角色？**

联系老板在「员工管理」页面编辑员工信息。

**Q: 可以添加更多工作室吗？**

可以。老板在「工作室管理」页面可创建多个工作室。
