核心表设计

👤 User (用户)
idUUID
studioId所属工作室 FK
roleOWNER/ADMIN/CS/COMPANION
username登录账号
passwordHash加密密码
isAuthorized老板是否已授权
secondPassword二级密码(bcrypt)
🎮 Companion (陪玩)
idUUID
userId关联 User FK
studioId所属工作室 FK
games擅长游戏 (JSON数组)
statusONLINE/OFFLINE/BUSY/IDLE
billingCode报账码 (唯一)
revenueShare分成比例 0.6 或 0.7
📋 Order (订单)
idUUID
typeNEW/RENEW/REPURCHASE/TIP
studioId工作室 FK
csUserId派单客服 FK
companionId接单陪玩 FK (nullable)
customerId客户 FK
dispatchTypePOOL/DIRECT (派单池/指定)
statusPENDING/GRABBED/CONFIRMED/DONE
amount金额 (Decimal)
gameName游戏名称
duration时长(小时)
customFields自定义字段 JSON
isOnline线上/线下
👥 Customer (客户)
idUUID
studioId工作室 FK
customerCode客户编号 (唯一)
wechatId微信号
companionId归属陪玩 FK
platform小红书/抖音/快手/B站/视频号
platformAccount平台账号名
consultDate咨询日期
wechatAddDate加微信日期
isAccountBanned账号是否封禁
isDeleted被客户删除
totalSpent累计消费 (冗余计算)
💻 CompanionPC (陪玩电脑)
idUUID
companionId关联陪玩 FK
agentVersionAgent 版本号
lastHeartbeat最后心跳时间
currentModeENTERTAINMENT/WORK
isThrottled是否限速中
throttleLimitKB限速值
💰 Transaction & Revenue

Transaction (交易)

id, orderId, amount, screenshotUrl
paymentMethod (WECHAT/ALIPAY), paidAt

RevenueDaily (日汇总)

date, studioId, companionId
newOrderCount/Amount, renewCount/Amount...

Expense (支出)

id, studioId, category, amount, date

计时与状态日志

追踪陪玩工作/娱乐时间，用于收入校验

CompanionTimeLog id, companionId, mode (WORK/ENTERTAINMENT), startedAt, endedAt, durationSeconds
PCOperationLog id, pcId, operation (SHUTDOWN/RESTART/THROTTLE), operatorId, createdAt
