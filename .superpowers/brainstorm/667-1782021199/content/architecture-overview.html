📦 推荐技术选型
🖥️ 前端React 18 + TypeScript + Ant Design
⚙️ 后端Nest.js (Express/Fastify)
🗄️ 数据库PostgreSQL + Prisma ORM
📡 实时Socket.IO (WebSocket)
⚡ 缓存Redis
🤖 AgentGo + WebView 本地 UI
📂 存储MinIO / 本地文件系统
